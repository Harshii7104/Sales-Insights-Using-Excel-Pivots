# Sales Insights Using Excel Pivots

## Project Overview
This project performs a comprehensive sales analysis using Excel Pivot Tables.
The goal is to identify revenue-driving categories, regional performance trends,
profitability insights, and underperforming business areas.

## Business Objective
• Analyze total sales and profit across categories and regions  
• Identify top-performing and underperforming regions  
• Evaluate profit margins to support data-driven business decisions  

## Dataset
Superstore Sales Dataset (publicly available sales data)

## Tools Used
• Microsoft Excel / Google Sheets  
• Pivot Tables & Pivot Charts  
• Slicers for interactivity  
• Conditional Formatting  

## Key Analysis Performed
• Total Sales by Category (ranked)  
• Sales by Region and Segment  
• Profit Margin Analysis  
• KPI Dashboard (Sales, Profit, Margin)  

## Key Insights
• Technology category contributes the highest share of revenue  
• West region is the top-performing region by sales  
• Furniture category shows lower profit margins  
• Corporate segment generates consistent revenue  
• South region requires business improvement

This project demonstrates practical Excel analytics skills, data summarization,
